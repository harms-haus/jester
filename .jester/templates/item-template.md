# Item Template

## Purpose

Template for creating item entities with consistent structure and metadata.

## Template Structure

```markdown
# {{ITEM_NAME}}

**Type:** item  
**Created:** {{CREATED_AT}}  
**Last Modified:** {{LAST_MODIFIED}}  
**Version:** {{VERSION}}

## Description

{{ITEM_DESCRIPTION}}

## Properties

- **Material**: {{MATERIAL}}
- **Size**: {{SIZE}}
- **Weight**: {{WEIGHT}}
- **Condition**: {{CONDITION}}

## Function

- **Primary Use**: {{PRIMARY_USE}}
- **Special Properties**: {{SPECIAL_PROPERTIES}}
- **Limitations**: {{LIMITATIONS}}

## Story Role

- **Owner**: {{OWNER}}
- **Key Scenes**: {{KEY_SCENES}}
- **Significance**: {{SIGNIFICANCE}}

## Metadata

- **Category**: {{CATEGORY}}
- **Rarity**: {{RARITY}}
- **Value**: {{VALUE}}
- **Origin**: {{ORIGIN}}
```
